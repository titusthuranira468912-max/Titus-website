TITUS THURANIRA | PERSONAL WEBSITE
===================================

This repository holds the foundation of my personal website.
It is a space where my work, skills, and ideas come together in one place.

The site is built to reflect growth, purpose, and consistency — both technically and personally.

--------------------------------------------------

WHAT YOU WILL FIND HERE

• A brief introduction about who I am  
• My skills, experience, and ongoing projects  
• Motivational and reflective content  
• A photo gallery capturing moments and ideas  
• A Bible study section using the King James Version (KJV)  
• A simple way to get in touch  

Everything is arranged to be clear, calm, and easy to explore.

--------------------------------------------------

HOW THE SITE IS BUILT

• Structured with clean HTML  
• Styled using custom CSS  
• Enhanced with JavaScript for interactivity  

The focus is on simplicity, readability, and future expansion.

--------------------------------------------------

VIEWING THE WEBSITE

• Clone or download this repository  
• Open `index.html` in any modern web browser  

The site may also be accessible online through GitHub Pages if enabled.
